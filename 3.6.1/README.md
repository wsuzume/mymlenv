# 3.6.1

## Usage

### Initial settings
```
$ pyenv install 3.6.1
$ pyenv local 3.6.1
$ python -m venv venv
$ . venv/bin/activate
$ pip install update pip
------------------------------------
$ pip install -r requirements.txt

or

$ pip install -r requirements.lock
------------------------------------
```

### Activate venv and start jupyter notebook
```
$ . venv/bin/activate
$ jupyter notebook
```

### Exit from venv
```
$ deactivate
```
